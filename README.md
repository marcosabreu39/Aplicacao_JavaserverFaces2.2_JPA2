# Agenda com JSF e JPA

## Tecnologias

* Arquitetura MVC;
* Maven;
* JavaServer Faces 2.2; 
* JPA 2 com Hibernate 5; 
* Enterprise Javabeans 3;
* Primefaces 5; 
* Banco de dados Oracle 11G;
* Servidor de Aplicação Wildfly 10; 
* IDE Eclipse Neon.
